If you need to uninstall the old version of the ProgRunner, use either of these commands:

C:\WINDOWS\Microsoft.NET\Framework\v1.1.4322\installutil /u ProgRunnerSvc.exe
 or
C:\WINDOWS\Microsoft.NET\Framework\v2.0.50727\installutil /u ProgRunnerSvc.exe

You can now install the new service:
cd C:\DMS_Programs\MultiProgRunnerSvc\
C:\WINDOWS\Microsoft.NET\Framework\v4.0.30319\installutil ProgRunnerSvc.exe
